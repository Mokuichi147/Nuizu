# 番手から糸径への変換

## 番手の定義

**メートル番手 (Nm)**: 1kg あたりの長さ（km）。数値が大きいほど糸は細い。

```
Nm = 1000 / tex
```

- **tex**: 1000m あたりの質量 (g)。数値が大きいほど糸は太い。
- **デニール (D)**: 9000m あたりの質量 (g)。`D = 9 × tex`

## 糸径 (mm) の計算式

糸を円柱と仮定し、断面積 `A = πd²/4`、長さ `l`、質量 `m = ρ × A × l` から導出:

```
tex (g/1000m) = ρ (g/cm³) × (πd²/4) (cm²) × 1000 (m) × 100 (cm/m) / 1
             = ρ × π × d² × 100000 / 4

→ d² = 4 × tex / (π × ρ × 1000000)

→ d (mm) = √( 4 × tex / (π × ρ × 1000000) )
```

`tex = 1000 / Nm` を代入:

```
d (mm) = √( 4 / (π × ρ × Nm) )
```

ここで ρ は繊維の密度 (g/cm³)。

出典: [Service Thread - How to Estimate Yarn Size Using Diameter](https://www.servicethread.com/blog/how-to-estimate-yarn-size-based-on-diameter) の逆算式 `tex = d² × ρ × 1000000 / (4π)` より導出。

## 刺繍糸の計算例

刺繍糸は一般的にポリエステル（ρ = 1.38）。

| 番手 (Nm) | 計算 | 糸径 (mm) |
|-----------|------|----------|
| #25 | √(4 / (π × 1.38 × 25)) | 0.192 |
| #50 | √(4 / (π × 1.38 × 50)) | 0.136 |
| #60 | √(4 / (π × 1.38 × 60)) | 0.124 |
| #80 | √(4 / (π × 1.38 × 80)) | 0.107 |
| #120 | √(4 / (π × 1.38 × 120)) | 0.088 |

## 注意事項

- 上記は単繊維を理想的な円柱と仮定した理論値。実際の糸は撚りや空隙があるため、実測径はこれより大きくなる。
- 刺繍では糸径がフィル行間隔に直結するため、理論値をそのまま使用している。

# JANOME JEF フォーマット仕様

JANOMEの独自刺繍ファイル形式。
座標は0.1mm単位の相対移動、Y軸反転。

出典: [EduTechWiki](https://edutechwiki.unige.ch/en/Embroidery_format_JEF), [pyembroidery](https://github.com/EmbroidePy/pyembroidery)

## ファイル構造

```
[ヘッダ 116バイト] [色インデックステーブル] [色タイプテーブル] [ステッチデータ]
```

全数値はリトルエンディアン (LE)。座標単位は **0.1mm**。

## ヘッダ (116バイト)

| オフセット | サイズ | 型 | 内容 |
|-----------|-------|-----|------|
| 0 | 4 | i32 | ステッチデータへのオフセット (`116 + 色数 × 8`) |
| 4 | 4 | i32 | フラグ（固定 `0x00000014`） |
| 8 | 14 | ASCII | 日時 (`YYYYMMDDHHmmSS` 形式、例: `20180712082429`) |
| 22 | 1 | u8 | バージョン文字（機種識別: m/n/o/p/q/r/s/t）またはゼロ |
| 23 | 1 | u8 | パディング (`0x20` or `0x00`) |
| 24 | 4 | i32 | 色数（色替え+1） |
| 28 | 4 | i32 | ポイント数（ENDを含む） |
| 32 | 4 | i32 | フープID |
| 36 | 4 | i32 | デザイン半幅 (0.1mm) |
| 40 | 4 | i32 | デザイン半高さ (0.1mm) |
| 44 | 4 | i32 | デザイン半幅（繰返し） |
| 48 | 4 | i32 | デザイン半高さ（繰返し） |
| 52 | 64 | 4×16B | フープ端距離（4種類のフープ分） |

合計: 116バイト

## フープ定義

| フープID | サイズ (mm) | 半幅 (0.1mm) | 半高さ (0.1mm) |
|---------|------------|-------------|---------------|
| 0 | 110 x 110 | 550 | 550 |
| 1 | 50 x 50 | 250 | 250 |
| 2 | 140 x 200 | 700 | 1000 |
| 3 | 126 x 110 | 630 | 550 |
| 4 | 200 x 200 | 1000 | 1000 |

### フープ端距離の計算

各フープに対し4つのi32値:

```
x_edge = hoop_half_width - design_half_width
y_edge = hoop_half_height - design_half_height

if x_edge >= 0 and y_edge >= 0:
    write(x_edge, y_edge, x_edge, y_edge)
else:
    write(-1, -1, -1, -1)  # デザインがフープに収まらない
```

## 色テーブル

ヘッダ直後に2つのテーブル:

### テーブル1: 色インデックス (`色数 × 4バイト`)

各エントリは i32 LE で JEF カラーパレットのインデックス。

### テーブル2: 色タイプ (`色数 × 4バイト`)

各エントリは i32 LE で固定値 `0x0D`。

## ステッチエンコーディング

座標単位: **0.1mm**、**最大ステップ: 127 (= ±12.7mm)**

### Y軸の反転

JEF形式ではY軸が反転: `dy_jef = -dy_cartesian`

### 通常ステッチ (2バイト)

```
[dx: i8] [-dy: i8]
```

dx, dy はそれぞれ符号付き8ビット（-127 ~ +127）。

### コマンドステッチ (4バイト)

```
[0x80] [コマンドID: u8] [dx: i8] [-dy: i8]
```

先頭バイト `0x80` (-128) がコマンドフラグ。

### コマンド一覧

| コマンド | バイト列 | 説明 |
|---------|---------|------|
| END | `0x80 0x10` | パターン終了（2バイトのみ、dx/dyなし） |
| 色替え | `0x80 0x01 dx (-dy)` | 糸色変更 |
| ジャンプ | `0x80 0x02 dx (-dy)` | 縫わずに移動 |
| トリム | `0x80 0x02 0x00 0x00` | 糸切り（ジャンプ+移動量ゼロ） |

### トリムの表現

トリムは「移動量ゼロのジャンプ」（`0x80 0x02 0x00 0x00`）で表現。
一般的に3回繰り返す。

### ステップ超過時の分割

dx, dy が ±127 を超える場合、複数コマンドに分割:

```python
while abs(dx) > 127 or abs(dy) > 127:
    step_x = clamp(dx, -127, 127)
    step_y = clamp(dy, -127, 127)
    write(step_x, step_y)  # or write_jump(step_x, step_y)
    dx -= step_x
    dy -= step_y
write(dx, dy)
```

## JEF カラーパレット

80色のインデックスカラー。主要な色:

| Index | RGB | 色名 |
|-------|-----|------|
| 0 | (0,0,0) | Black |
| 1 | (0,0,255) | Blue |
| 2 | (0,136,0) | Green |
| 3 | (255,0,0) | Red |
| 4 | (128,0,128) | Purple |
| 5 | (255,255,0) | Yellow |
| 6 | (192,192,192) | Gray |
| 7 | (255,128,0) | Orange |
| 11 | (255,255,255) | White |
